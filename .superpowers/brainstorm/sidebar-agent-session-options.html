lc-agent 侧边栏 Agent / 会话层级方案

已按你的选择收敛为：`双层卡片型`、卡片头突出 `Agent 名称 + 会话数`、当前状态时 `Agent 卡片头 + 会话标题` 同时高亮、并且展开状态记忆用户上次操作。 下面 3 套方案都兼容黑白主题，差别主要在产品感、密度和层级强弱。

推荐标准

  • Agent 和会话一眼分层
  • 当前所在 Agent 足够明确
  • 暗色和浅色都不糊
  • 手机缩窄后仍然稳

A. 静稳卡片

推荐

每个 Agent 是一张安静的导航卡片。卡片头只讲 `名称 + 会话数`，当前 Agent 整张卡片轻发光，当前会话是卡内一枚高亮条。层级最清楚，也最耐看。
